SSR，全称为 ShadowsocksR（下文简称为SSR） 来历：

ShadowsocksR是breakwa11发起的Shadowsocks分支，在Shadowsocks的基础上增加了一些数据混淆方式，称修复了部分安全问题并可以提高QoS优先级。后来贡献者Librehat也为Shadowsocks补上了一些此类特性，甚至增加了类似Tor的可插拔传输层功能。

ShadowsocksR开始时曾有过违反GPL、发放二进制时不发放源码的争议，不过后来ShadowsocksR项目由breakwa11转为了与Shadowsocks相同的GPL、Apache许可证、MIT许可证等多重自由软件许可协议。

1. 电脑翻墙
电脑翻墙通常使用VPN、代理服务器或其他工具，帮助用户访问被限制的网站和服务。常用的翻墙工具包括VPN软件、Shadowsocks、V2Ray等。

2. 网络翻墙软件
网络翻墙软件能够帮助用户绕过地理限制和网络审查。常见的软件包括：

VPN：如ExpressVPN、NordVPN，提供加密和匿名访问。
代理软件：如Shadowsocks和V2Ray，适合需要灵活配置的用户。
浏览器扩展：如SwitchyOmega和Ultrasurf等。
3. 谷歌浏览器如何翻墙
在谷歌浏览器中翻墙可以通过以下方法：

使用VPN扩展：在Chrome网上应用店中搜索并安装VPN扩展，登录后即可使用。
设置代理：在浏览器的网络设置中手动配置代理服务器。
使用翻墙软件：在计算机上运行翻墙软件，通过它来访问谷歌浏览器。
4. VPN免费翻墙软件
市场上有一些免费的VPN软件，虽然它们可能有限制（如速度、流量、服务器选择等），但可以作为尝试。常见的免费VPN包括：

ProtonVPN：提供免费的基本套餐。
Windscribe：有免费账户，限制流量。
TunnelBear：使用简单，但每月有流量限制。
5. 翻墙谷歌
访问谷歌时，如果受到限制，可以使用翻墙工具（如VPN或代理）来解除限制，确保你能够顺畅访问谷歌搜索和其他服务。

6. 苹果翻墙
在苹果设备上翻墙，可以通过以下方法：

安装VPN应用：从App Store下载VPN应用，注册并登录。
使用代理设置：在“设置”中找到“Wi-Fi”，点击所连接的网络，输入代理服务器信息。
配置翻墙软件：如Shadowsocks等，根据提供商的说明进行设置。
7. 翻墙代理
翻墙代理服务器是一种通过中间服务器转发请求的方式。使用代理时，用户的请求首先发送到代理服务器，然后由代理服务器访问目标网站，最终返回结果给用户。常用的翻墙代理包括HTTP代理和SOCKS5代理。

8. 翻墙路由器
翻墙路由器是一种预装了翻墙软件的路由器，可以为整个家庭或办公室提供翻墙服务。选择翻墙路由器时，可以考虑以下几点：

兼容性：确保路由器支持所需的翻墙软件。
速度：选择高性能的路由器，以保证翻墙速度。
设置方便性：一些路由器提供易于使用的界面和设置向导。
